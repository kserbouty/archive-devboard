# Devboard (Prototype)

## Installation

### Requirements

- Apache Server >=2.4
- PHP >=7.4 with Composer
- MySQL >=8.0

1째 Import the database.sql in your environment

2째 Set your database in the config.ini

3째 Install the dependencies

```bash

```

3째 Run your server on <http://localhost:8000>

